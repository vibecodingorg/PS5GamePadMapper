# PS5GamePadMapper

PS5GamePadMapper 是一款专为开发者及小团队打造的 macOS 平台输入映射工具。核心目标是实现在 macOS 上高效、低延迟地将 PS5（DualSense）手柄输入映射为键盘与鼠标事件，并提供强大的宏与脚本支持，主要服务于开发者的游戏测试、技能连招调试与输入压力测试需求。

## 🎯 核心功能

- **基础硬件通讯**: 稳定识别 PS5 (DualSense) 手柄，支持 USB 与 Bluetooth 双连接方式。
- **动态按键映射**: 
  - 支持将手柄按钮（Button）映射至键盘组合键、鼠标按键及滚轮。
  - 支持将模拟轴（Axis，包括摇杆、扳机）映射至 WASD/方向键或相对鼠标移动（支持死区与灵敏度调节）。
- **高阶宏系统 (Macro)**: 提供针对开发者设计的高灵活性宏系统，支持顺序、循环（Turbo）、切换（Toggle）及并行宏。
- **可编程脚本 (Script)**: 暴露最小集 API（如事件注入与精确延时等），允许开发者根据条件使用脚本直接干预复杂输入状态。
- **配置管理 (Profile)**: 支持在本地以 JSON 格式快速保存配置项，支持快速调试克隆，并能够实现多套映射预案即时切换。

## 🛠 技术栈说明

- **UI 设计**: 仅供内部及开发自用，基于 SwiftUI 实现极简调试面板工具栏，不注重商业化外壳。
- **底层驱动**: 
  - 设备读取层：通过 `IOHIDManager` 进行事件监听。
  - 事件注入层：通过 `CGEventPost` 进行 macOS 系统级事件模拟。
  - 调度实现：采用高优先级串行队列及精准定时器，确保单次输入延迟目标 `< 30ms`，并在高频宏输入情况下的稳定性。

## ⚙️ 系统与权限要求

- **操作系统**: macOS 环境。
- **系统授权**: 
  - 程序必需获准 **辅助功能 (Accessibility)** 权限，这是 macOS 事件注入的先决条件。
  - 如果需要监听全局输入状态，则同样需准允 **输入监听 (Input Monitoring)**。

## 🚀 使用指南

1. **构建与运行**: 使用 Xcode 编译运行内层 `PS5GamePadMapper` 工程源码。
2. **授权**: 首次启动请根据提示打开系统设置，授予相关模拟点击/按键权限。
3. **映射配置**: 连接手柄后，可在主界面点击对应按钮唤出映射配置面板，添加 `Key` / `Mouse` / `Macro` 型规则，修改即时生效无需重启。
